Agent Workflow Demo

一个用于展示任务拆解、工作区扫描、报告生成和结果校验流程的示例项目。

看项目说明 看示例输出

主要功能

  • 将输入目标拆解为执行步骤
  • 扫描目标目录并提取文件信息
  • 生成 Markdown 格式的分析报告
  • 输出 trace 文件用于记录执行结果

执行流程

输入目标后，程序依次执行任务拆解、目录扫描、内容整理、报告生成和结果校验。

产出文件

  • `output/report.md`：Markdown 报告
  • `output/trace.json`：结构化执行记录
  • `demo_workspace/`：示例工作区
  • `web/`：静态展示页面

运行方式

通过命令行传入目标和工作区路径后，程序执行扫描、整理和输出流程，并在 `output/` 目录中生成结果文件。
